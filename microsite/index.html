CSS Snap Points Example

CSS Scroll snap points lets you create easy to scroll regions in which scrolling is snapped to a series of points. Currently supported in FireFox, Safari (prefixed) and Opera (prefixed). Check the current support on caniuse.

Horizontal Scroll Gallery - Different image sizes

Below we have created a gallery where images of all different sizes and shapes are displayed horizontally

With scroll snapping we can make it so that as you scroll the container, the browser will stop in the center of the slider and also the center of the image (if there is enough space allowed)
